1. Pendahuluan

Dokumentasi ini akan memandu Anda melalui proses instalasi dan konfigurasi Maltego di sistem operasi Windows.

Maltego adalah sebuah platform investigasi berbasis open-source intelligence (OSINT) yang dikembangkan oleh Paterva. Tools ini banyak digunakan oleh praktisi keamanan siber, peneliti digital, hingga penyidik forensik karena kemampuannya dalam mengumpulkan, memetakan, dan menganalisis informasi dari berbagai sumber publik di internet.

2. Langkah 1: Download Maltego

Sebelum memulai, Anda perlu mengunduh Maltego melalui tautan berikut:

Unduh di Sini
  1. Operating System Windows
  2. Extension .exe + Java (x64)
  3. Extension .exe(jika sudah menginstal Java)
  4. tekan unduh

3. Langkah 2: Instalasi Java

Menginstal Java, jika pada device belum terinstal javaapt:

  1. Mencari file instalasi Maltego yang sudah di unduh sebelumnya
  2. klik dua kali file tersebut

    3. Gambar 2.1: Tampilan saat membuka file instalasi.

  3. Klik "OK".

    4. Gambar 2.2: Tampilan pembuka saat instalasi java.

  4. Klik "next".

    5. Gambar 2.3: Tampilan custom setup.

  5. Atur lokasi aplikasi(opsioanl). Tekan "Nest" jika tidak ada yang ingin diubah.

    6. Gambar 2.4: Tampilan instalasi akhir java.

  6. Jika tampilan sebagai berikut, instalasi yang Anda lakukan sudah berhasil. Klik "Finish"

    7. Gambar 2.5: Tampilan instalasi berhasil.

4. Langkah 3: Instalasi Maltego

Jika Anda sudah menginstall java sebelumnya maka tampilan secara otomatis berganti menjadi instalasi maltego.

Gambar 3.1: Tampilan pembuka saat instalasi Maltego.

  1. Klik "Next"

    2. Gambar 3.2: Mengatur user.

  2. Pilih sesuai keinginan, untuk semua user atau hanya user saat ini. Klik "Next"
  3. Tentukan lokasi aplikasi akan disimpan. Klik "Next"

    4. Gambar 3.3: Memilih lokasi instalasi.

  4. Klik "Install"

    5. Gambar 3.4: Memilih folder start menu.

  5. Menunggu proses instalasi selesai.

    6. Gambar 3.5: Proses instalasi.

  6. Selamat Anda sudah berhasil menginstal maltego, jika tampilan sudah seperti ini. Klik "Finish"

    7. Gambar 3.6: Berhasil menginstal maltego.
